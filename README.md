## About The Project

![main](https://user-images.githubusercontent.com/80919963/177055500-7f139c4e-8914-4bee-a5c4-94fffc927ffa.PNG)


This project was developed for my diploma thesis defense. March 2022, BNTU. **It is my first ever project built with .NET.**

The diploma project was defended with the top grade of **10 out of 10**.

Project features:
* A lightweight, custom CMS
     - Add / Remove / Edit products
     - Add / Remove / Edit blog posts
     - Process customer orders
     - Add / Remove users with CMS access
* CMS access levels
     - Administrator
     - Manager
* Sorting, search, and filtering
* Product comparison
* Wishlist (favorites)
* Shopping cart with payment via the Stripe service
* Customer reviews and ratings
